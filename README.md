## 使用技術
- React Native
- JavaScript
- Expo

## 機能
- 色を当てるゲーム
- ボタンアニメーション
- ハプティクス（振動）
- 制限時間機能

## 工夫した点
- 状態管理を意識して不要なレンダリングを減らした
- コンポーネント化により再利用性・保守性を向上
- ボタンを押したときの心地よい操作感を追求
